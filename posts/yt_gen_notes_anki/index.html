Python & OpenAI - Generate Notes + Anki Flashcard Deck From Youtube Video

April 3, 2024

  • This script uses pytube, genanki and OpenAI to produce notes and a deck of flashcards. The code is available on my Github here –> https://github.com/elmcgu/YT-Notes-Flashcard-Gen

  • I created this script to help process free learning material on Youtube for CompTia Network+ Cert.

  • It was an interesting learning project - regex for file renaming and cleaning json, parsing json into python dictionary, learning to interact with OpenAI API and prompt engineering.

  • I have more variations of the script including using local AI models and iterating through playlists which I will upload to Github when I have a chance.

  • You just need to provide link to youtube video and and give name to the folder where you want to save.

  • The outputs saved to folder are .mp3 audio, .txt transcript, notes in markdown format and an .apkg file that can be imported into Anki desktop app.

  • In order to use this you must have an OpenAI key and add it as an environment variable in your shell by adding to .bashrc file(linux). Windows and MacOS instructions –> https://platform.openai.com/docs/quickstart?context=python

  • Both the pytube and OpenAI modules must be installed with pip before running script.

  • The AI has the personality of Marvin the Paranoid Android from Hitchiker’s Guide to the Galaxy hence the odd and nihilistic commentary when running the program.
